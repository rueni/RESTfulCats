# RESTfulCats
A RESTful API in Sinatra dedicated to cats.

**GET: List** <br>
get '/api/cats'

**GET: By id** <br>
get '/api/cats/:id'

**POST: create a new record** <br>
post '/api/cats'

**UPDATE a record by id** <br>
patch 'api/cats/:id'<br>
put 'api/cats/:id'

**DELETE a record** <br>
delete '/api/cats/:id'
